## 📝 문제별 정리 방식

각 문제는 다음과 같은 구조로 정리됩니다:

### 폴더 구조
```
problem-name/
├── solution.py     # 솔루션 코드
├── README.md       # 문제 분석 및 해설
└── test.py         # 테스트 케이스 (선택사항)
```

### README.md 포함 내용
- 문제 설명 및 링크
- 접근 방법 (Approach)
- 시간 복잡도 (Time Complexity)
- 공간 복잡도 (Space Complexity)
- 핵심 아이디어
- 배운 점
